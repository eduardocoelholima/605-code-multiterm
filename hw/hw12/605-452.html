Homework Description:

You have to create a multi threaded program which prints out numbers in increasing order. Each thread can only print the number it was give to its constructor. The program shown in class which prints out the sequence 1010101... would be an example, if 2 threads would be used.

My program produces the following output:

          java PrintNumbers -nThreads 3 -soOften 2
          # round = 1   -------
          0
          1
          2
          # round = 2   -------
          0
          1
          2

You do not have to verify that the arguments are valid.

Requirements:

• the number of threads must be specified via command line argument.

• the number of rounds must be specified via command line argument.

• you can not use busy loops.

• wait and notify must be used to achieve the desired output.

• the program has to terminate after each thread printed its id ’soOften’ times.

Your output does not have to be identical to mine, but similar.
