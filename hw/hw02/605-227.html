A few requirements must be considered:

• The user guess needs to be 5 character long. The user guess can contain characters, numbers, etc. The program must terminate if the user guess is not 5 characters long.

• The file containing the words to guess cannot have any duplicates. The program must terminate if the file containing the words has duplicates.

• Your program cannot assume the # of words in the words to guess file. In other words, you cannot hard code the length of the array which is used to stores the words to guess.

• For every round a random word for the words to guess has to be selected and removed from the list of the words to guess.

• You cannot convert a string to an array.

• Only methods provided in the example of methods from the string class

• You have to document additional assumptions you made.
