Explanation:

Given is a file like:

aa

The out put to show where the characters from the first string end up in the 2. string.

% java Transform
ab a ---> baa
    0(a) -----> (1)
    1(b) -----> (0)
    = =
    3(a) -----> (2)
baa ---> ab a
    0(b) -----> (1)
    1(a) -----> (0)
    2(a) -----> (3)

A human explained solution would be:

ab a ---> baa
    0(a) -----> (1)    # a from position 0 ends up at position 1
    1(b) -----> (0)    # b from position 1 ends up at position 0
    = =                # not relevant
    3(a) -----> (2)    # a from position 3 ends up at position 2 and not 1

Your output does not have to be identical to mine, but similar.
