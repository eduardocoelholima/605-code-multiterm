Explanation:

Given is a file, and every line consists of one phrase. An example would be:

% cat example.txt
William Shakespeare
A decimal point
I am a weakish speller
I’m a dot in place

You have determine if two lines are anagrams of each other. Two phrases are anagrams if they the second phrase be created by transposing the letter of the first phrase. Punction marks and white space are not treated as letters.

Your Work:

Design and implement a program which can group all lines of a file to anagrams.

Example:

An example of a solution execution:

This is a execution of a possible solution:

$ java AnagramGrouper

Two phrases have been identified as anagrams once. Keep in mind, if A is an anagram to B, then B is an anagram to A. The following output would be incorrect:

-A decimal point-   -I’m a dot in place-

========
