33.1. Homework 13.1 (10 Points)

You have to design and implement a PingServer and a Ping Client. The round trip time has to be measured in nano seconds. You have to use try-ressource statements.

Ping Server command line syntax: [-port portId]

Example: java PingServer

-port 3456.

Ping Client command line syntax: [-port portId] [-count n] [-hostName hostName]

Example: java Ping -port 3456 -count 5 -hostName 10.118.56.117

You can set reasonable default values for all command line arguments.

This is an example of a solution:

% java PingServer   -port 3456 &
Listening on port: 3456
% java Ping         -port 3456 -count 5
round-trip to localhost = 1183583 ns.
round-trip to localhost = 218166 ns.
round-trip to localhost = 134702 ns.
round-trip to localhost = 126412 ns.
round-trip to localhost = 198108 ns.

Your output does not have to be identical to mine, but similar.
