Homework Description:

This is a modification of hw 2.1 (Wordle). The functionality of the player, logic server, and dictionary server must be executed in different JVM’s and the communication between server and client has to be executed via Sockets. You have to use try-ressource statements.

Requirements:

• The dictionary server reads the words file.

• The dictionary server selects the word to guess.

• The player reads the guess from the user

• The player sends the input to the logic server and receives the word to display from the logic server

• The logic server sends the result of the analyzed guess back to the player.

• The logic server decides if the word has been guessed correctly.
