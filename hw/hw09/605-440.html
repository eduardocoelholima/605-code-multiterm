Homework Description:

Explanation:

The objective of this hw is to understand how to use Try-with-resources and basic I/O files.

Your Work:

You have to design and implement a version of the UNIX command wc.

Your implementation has to support:

• -l print the newline counts

• -c print the byte counts

• -m print the character counts

• -L print the maximum display width

• - the input file is stdin

By default all flags are true, and arguments given set the flag(s) to false.

The syntax for using the java wc version is: java Wc [-c] [-l] [-m] [fileName|-]

A fileName == ’-’ means the program need to read from stdin.

Requirements:
