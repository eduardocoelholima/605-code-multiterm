Homework Description:

Explanation:

The objective of this hw is to understand how to use Try-with-resources and basic I/O files.

Your Work:

You have to design and implement a version of the UNIX command dd.

Your implementation has to support:

• if=Input_FILE

• of=Output_FILE

• bs=BYTES

• skip=N

FILE="-’ means reading from stdin or writing to stdout.

Requirements:

• You can only use classes discussed in classes

• Input_FILE can not be stdin, why?
Output_FILE culd be stdin, indicated by ’-’.

• You have to use Try-with-resource A test produces the following output:

          % java DiskDump if="12345.txt" of="-" skip=3 bs=1 > out.txt
          cat out.txt
          45
          % javac -Xlint  DiskDump.java

          45

Your output does not have to be identical to mine, but similar.
