Homework Description:

Take a look at the following program:

 1      public class T1 extends Thread    {
 2          static Object o = new Object();
 3          int id;
 4
 5          public T1(int id)   {
 6              this.id = id;
 7              o = new Object();
 8          }
 9          public void ring () {
10              o.notify();
11          }
12          public void run () {
13              synchronized ( o ) {
14                      try {
15                              ring();
16                      } catch (  Exception e ) {
17                              System.out.println(e );
18                              System.exit(0);
19                      }
20              }
21          }
22          public static void main (String args []) {
23              new T1(1).start();
24              new T1(2).start();
25              System.out.println("Done");
26          }
27      }

The execution of this code could produce the folloing output
